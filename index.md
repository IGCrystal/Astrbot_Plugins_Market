## AstrBot Plugins Market (a page for practice)

During the development of the official AstrBot plugin market, the application was built entirely with Vue.js, serving solely as a plugin browsing interface. Following my departure from the AstrBot team, I forked the repository into the neo-life organization and established it as an independent project to migrate the stack to Nuxt.js. In this new environment, I actively executed significant SEO optimizations and explored enhancements like intelligent recommendations. The process yielded valuable insights.
